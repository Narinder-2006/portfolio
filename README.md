# Personal Portfolio

A highly responsive, data-driven personal portfolio website built to showcase my projects, skills, and experience in Machine Learning, Deep Learning, and Software Engineering.

## 🚀 Live Demo
[Link to your live Vercel website will go here once deployed]

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Design Tokens & Glassmorphism), Vanilla JavaScript
* **Mapping:** Leaflet.js (Interactive location rendering)
* **Architecture:** Data-driven UI via a single configuration object (`portfolioData.js`)

## ✨ Features
* **Modular Data Structure:** All content is separated from the UI logic, making it incredibly easy to update projects, experience, and skills without touching the HTML.
* **Dynamic Animations:** Custom CSS keyframes, intersection observers for scroll-reveals, and glowing hover states.
* **Responsive Design:** Fluid typography and grid layouts that adapt seamlessly from mobile devices to ultrawide monitors.
